Aprendendo sobre variaveis

variavel var e let: pode, ser alteradas para novos valores

variavel const: não pode ser alterada na sua classe de 1º estancia. Para alterar uma const, devemos modificar os atributos e metodos de uma classe do objeto.
Exemplo:
const pessoa = {nome: 'victor', idade: 30}.
nome e idade são clases de 1º instancias do objeto, não podem ser alterados.
const = pessoa.nome = Luana
const = pessoa.idade = 27.
São atributos e metodos que podem ser alterados
