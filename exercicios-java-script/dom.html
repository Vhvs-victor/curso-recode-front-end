Aprendendo sobre DOM (Document Object Model)

Window

  • window.open (Abrir uma pagina do navegador);

  • window.close (Fechar uma janela);

  • window.location (Definir qual site sera acessado);

Document

Document: representa todo documento HTML e seus metodos e atributos. Pelo document, podemos acessar elementos especificos usando ID, NAME, TAG, CLASS.

  • getElementById('');
  • getElementsByName('');
  • getElementsByTagName('');
  • getElementsByTagClassName('');

  • Acessar o titulo do documento (document.title);
  • Acessar o URL do documento (document.url);

Criar um elemento HTML pelo Java Script (document.createElement);
# O object element, representa elementos HTML, tais como: <body>, <div> ,<h1>, <p> e muitos outros.

  • Modificar sua classe CSS(elemento.className);
  • Modificar o conteudo(elemento.innerHTML); ou (document.outerHTML);
  • Saber quem são os elementos filho(document.children);
  • Ter acesso aos seus atributos(document.attributes);

  • window.innerWidth
    • Tamanho interno da janela do navegador
  • screen.width
    • Tamanho da resolução da tela do monitor
  • window.onrezi
    • Evento de redimencionamento da janela do navegador
  • window.scroll(eixo x/horizontal, eixo y/vertical)
    • window.scroll(0,window.innerWidth Rolagem automatica no eixo y, enquanto redimencionar a janela do navegador, sem rolagem no eixo x, 0.
  • document.write
    • Substitui todo documento HTML pelo texto de referencia do document.write('Olá, Mundo!');

Eventos em DOM

Eventos: São gatilhos que são disparados sempre que algo acotecer conforme a interação do usuario.

  • ONCLICK: no clique, quando o usuairio clica em determinado elemento.
  • ONMOVEMOUSE: Quando o usuario move o mouse por cima de um determinado elemento.
  • ONCHANGE: É disparado quando um elemento HTML é alterado.
  • ONMOVEOUT: Quando o usuario move o mouse para fora do elemento.
  • ONKEYDOWN: Quando o usuario pressiona alguma tecla do teclado.
  • ONLOAD: Quando o navegador termina de carregar a pagina
